# Supabase Setup Guide for Louisa's Perfumes

## Step 1: Get Your API Keys

1. Go to: https://bqnepoxtsfqyuhsvxzby.supabase.co/project/settings/api
2. Copy these keys and save them:
   - **SUPABASE_ANON_KEY** (Anonymous/Public Key)
   - **SUPABASE_SERVICE_ROLE_KEY** (Service Role Key) ⚠️ Keep this secret!

## Step 2: Create the Products Table

1. Go to your Supabase dashboard → **SQL Editor**
2. Click **New Query** and paste this:

```sql
-- Create products table
CREATE TABLE products (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  name VARCHAR(255) NOT NULL,
  manufacturer VARCHAR(255) NOT NULL,
  ml INTEGER NOT NULL,
  price INTEGER NOT NULL,
  description TEXT NOT NULL,
  notes TEXT[] DEFAULT '{}',
  image_url VARCHAR(500),
  in_stock BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Create indexes for better query performance
CREATE INDEX idx_products_name ON products(name);
CREATE INDEX idx_products_in_stock ON products(in_stock);

-- Enable RLS (Row Level Security)
ALTER TABLE products ENABLE ROW LEVEL SECURITY;

-- Allow anyone to read products
CREATE POLICY "Allow public read" ON products
  FOR SELECT USING (true);

-- Only authenticated users can insert (we'll verify admin role in API)
CREATE POLICY "Allow authenticated insert" ON products
  FOR INSERT WITH CHECK (auth.role() = 'authenticated');
```

3. Click **Run** and wait for success message

## Step 3: Set Up Storage for Images

1. Go to **Storage** → **Create new bucket**
2. Name it: `product-images`
3. Make it **Public**
4. Click **Create bucket**

## Step 4: Enable Email Auth

1. Go to **Authentication** → **Providers**
2. Make sure **Email** provider is enabled (it should be by default)
3. Go to **Email Templates** and customize signup/confirmation if needed

## Step 5: Add API Keys to Vercel

1. Go to your Vercel dashboard → Project settings → Environment Variables
2. Add these variables:
   - `SUPABASE_URL` = `https://bqnepoxtsfqyuhsvxzby.supabase.co`
   - `SUPABASE_ANON_KEY` = your anonymous key
   - `SUPABASE_SERVICE_ROLE_KEY` = your service role key (⚠️ keep secret!)
   - `GEMINI_API_KEY` = your Gemini API key

3. Click **Save & Deploy**

## Step 6: Test It

1. Go to your deployed site
2. Try uploading a product as admin
3. Check if it appears in the product list
4. Ask the AI chatbot about it

## Troubleshooting

- **"Missing SUPABASE_URL"** → Check Vercel environment variables
- **"Failed to fetch products"** → Make sure the `products` table exists
- **"Invalid token"** → Supabase Auth token expired, need to re-login
- **Images not showing** → Check the image URL format and storage bucket permissions

## API Endpoints

- `GET /api/products-db` — Get all products
- `POST /api/auth` — Login/Signup (body: `{action, email, password}`)
- `POST /api/admin-products` — Create product (requires valid auth token)
- `POST /api/recommend` — Get AI recommendation
